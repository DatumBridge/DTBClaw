# Справочник команд (Русский)

Это первичная локализация Wave 1 для быстрого поиска CLI-команд OctoClaw.

Оригинал на английском:

- [../../commands-reference.md](../../commands-reference.md)

## Когда использовать

- Поиск команд по типовой задаче
- Проверка флагов и границ поведения
- Сверка ожидаемого результата при отладке

## Правило

- Имена команд, флагов и ключей конфигурации сохраняются на английском.
- Финальная спецификация поведения — в английском оригинале.

## Последнее обновление

- `octoclaw gateway` поддерживает `--new-pairing`: флаг очищает сохранённые paired-токены и генерирует новый код сопряжения.
- В английский оригинал добавлены поверхности миграции OpenClaw: `octoclaw onboard --migrate-openclaw`, `octoclaw migrate openclaw` и агентный инструмент `openclaw_migration` (полная локализация этих пунктов в процессе).
